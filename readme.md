## 📰 SNN Brasil - Site de Notícias

Este projeto é um **portal de notícias fictício** desenvolvido em **HTML e CSS**  
O objetivo é praticar estruturação de páginas web, estilização com Flexbox.

## ⚙️ Funcionalidades
- Menu de navegação com seções de notícias.
- Cabeçalho com destaques do dia.
- Cards de notícias principais com imagens.
- Seção lateral para notícias adicionais.
- Rodapé com editorias e informações institucionais.
- Estilização com **Flexbox** e efeitos de hover.

├── index.html        **Página principal**
├── styles/
│   └── index.css     **Estilos da página**
└── img/              **Imagens utilizadas**

## 💻 Tecnologias Utilizadas
- **HTML5** para marcação semântica
- **CSS3** para estilização
- **Flexbox** para layout responsivo
